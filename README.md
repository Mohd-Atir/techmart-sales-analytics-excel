# TechMart Sales Analytics Dashboard

## Project Overview
End-to-end Excel analytics project built on a fictional 
e-commerce dataset (TechMart). Demonstrates the full 
Data Analyst workflow from raw CSV ingestion to 
executive dashboard.

## Tools Used
- Microsoft Excel 365
- Power Query (ETL & data modelling)
- Pivot Tables & Pivot Charts
- Slicers (interactive filtering)
- Advanced Formulas (SUMPRODUCT, XLOOKUP, COUNTIFS)

## Dataset
| Table | Rows | Description |
|---|---|---|
| orders | 500 | Transactional order data |
| customers | 500 | Customer master data |
| products | 500 | Product catalogue |
| sales_reps | 500 | Sales rep deal performance |

## Key Metrics Analysed
- Total Revenue: ₹4.22 Cr (2022–2024)
- Profit Margin: 17.9%
- Sales Win Rate: 45.8%
- Top Region: North (₹1.46 Cr)
- Top Sales Rep: Anita Banerjee

## Dashboard Features
- 8 live KPI cards
- 4 interactive charts
- 4 slicers (Region, Status, Year, Channel)
- Data quality audit sheet
- Full KPI engine (93 metrics)

## How to Use
1. Open the .xlsx file in Excel 365
2. Go to the Dashboard sheet
3. Use slicers on the right to filter by 
   Region / Year / Status / Channel
4. All KPI cards and charts update automatically
